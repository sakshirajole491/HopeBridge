# 🌉 HopeBridge

A **full-stack web application** built with **Java Spring Boot** that connects individuals in need with people willing to help — enabling efficient request management and community-driven support.

---

## 🎯 Project Highlights

* Designed and developed a **scalable Spring Boot application** following MVC architecture
* Implemented **secure user authentication & session handling**
* Built **RESTful APIs** to manage user requests and resources
* Integrated **MySQL database** for persistent data storage
* Developed a **dynamic UI using Thymeleaf** for seamless user interaction
* Improved usability with **clean navigation and responsive design**

---

## 🚀 Features

* 🔐 User Registration & Login System
* 📝 Create, update, and manage help requests
* 🔍 Browse and explore available support/resources
* 📊 Structured backend with REST API endpoints
* 💡 Clean and intuitive UI for better user experience

---

## 🛠️ Tech Stack

* **Backend:** Java, Spring Boot
* **Frontend:** Thymeleaf, HTML, CSS, JavaScript
* **Database:** MySQL
* **Build Tool:** Maven

---

## 🏗️ Architecture Overview

* Followed **MVC (Model-View-Controller)** design pattern
* Controller layer handles request routing
* Service layer manages business logic
* Repository layer interacts with database

## ⚙️ Setup & Run Locally

### Prerequisites

* Java 8 or higher
* Maven
* MySQL installed and running

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/sakshirajole491/HopeBridge.git
   ```

2. Navigate to the project folder:

   ```bash
   cd HopeBridge
   ```

3. Configure database in `application.properties`

4. Run the application:

   ```bash
   mvn spring-boot:run
   ```

5. Open in browser:

   ```
   http://localhost:8080
   ```
